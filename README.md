
# Data Mining Pelanggan dengan Streamlit

## Cara Menjalankan

1. Install Python
2. Install library:
   pip install -r requirements.txt

3. Jalankan Streamlit:
   streamlit run app.py

4. Upload file dataset_pelanggan.csv

## Fitur
- Preview dataset
- Statistik deskriptif
- Histogram pendapatan
- Grafik kategori pelanggan
